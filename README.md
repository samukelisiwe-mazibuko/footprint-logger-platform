Installation
Backend Setup
cd backend
npm install
npm start
Frontend Setup

Open the HTML files inside the frontend/pages folder in your browser.

Environment Variables

Create a .env file inside the backend folder and add:

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
